# Universify Project

A project made to lookup universities at a quick and ease manner.
Live: https://university-app.vercel.app/

<img src="https://kobi.lol/assets/universify.webp" alt="Universify" />

## Technology Used

- ReactJS's NextJS ( SSR )
- TailwindCSS
- CSS
- HTML
- Javascript

## Notable Features

- O(n) Time Complexed algorithms sorting
- Country Filtering
- Lookup by name / country
- Redirect to website on click.
